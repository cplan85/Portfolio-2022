Event-Zorro

Event Zorro is a MEAN-Stack platform that allows users to search for events in their area. This App mainly utilizes Ticketmaster’s Discovery API, MapBox Geocoding Places API, and Mapbox’s GL JS library to allow users to retrieve local events, and then search for events around the world. The Back-end is developed in Node and Express with MongoDB as the database allows users to employ CRUD operations on three collections of Users, Events, and Maps...
